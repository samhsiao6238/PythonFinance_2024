# MacOS 指令 tree

_在 Linux 是預設的_

<br>

## 安裝 Homebrew

1. 查詢版本

    ```bash
    brew -v
    ```

<br>

2. 假如要安裝 Homebrew

    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```

<br>

3. 安裝。

    ```bash
    brew install tree
    ```

<br>

## brew 報錯

1. 修理。

    ```bash
    brew tap --repair
    ```

<br>

2. 更新。

    ```bash
    brew update
    ```

<br>

3. 檢查。

    ```bash
    brew doctor
    ```
