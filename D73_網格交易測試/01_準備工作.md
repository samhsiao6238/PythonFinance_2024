# 從頭開始

<br>

## 準備工作

_開啟終端機進行操作_

<br>

1. 建立虛擬環境。

    ```bash
    deactivate && cd ~/Documents/PythonVenv && python -m venv envStock
    ```

<br>

2. 編輯環境變數。

    ```bash
    code ~/.zshrc
    ```

<br>

3. 啟動虛擬環境。

    ```bash
    source ~/.zshrc
    ```

<br>

4. 更新 pip。

    ```bash
    pip install --upgrade pip
    ```

<br>

5. 檢查當前套件。

    ```bash
    pip list
    ```

<br>

6. 建立並進入專案。

    ```bash
    mkdir -p ~/Desktop/_exStock_ && cd ~/Desktop/_exStock_ && code .
    ```
