# 從頭開始

<br>

## 準備工作

_開啟終端機進行操作_

<br>

1. 先退出當前環境。

    ```bash
    deactivate
    ```

<br>

2. 建立新的虛擬環境 `envStock`，這裡使用的路徑是在 `~/Documents/PythonVenv`。

    ```bash
    cd ~/Documents/PythonVenv && python -m venv envStock
    ```

<br>

3. 編輯環境變數。

    ```bash
    code ~/.zshrc
    ```

<br>

4. 啟動虛擬環境。

    ```bash
    source ~/.zshrc
    ```

<br>

5. 更新 pip。

    ```bash
    pip install --upgrade pip
    ```

<br>

6. 檢查當前套件。

    ```bash
    pip list
    ```

<br>

7. 建立並進入專案。

    ```bash
    mkdir -p ~/Desktop/_exStock_ && cd ~/Desktop/_exStock_ && code .
    ```

<br>
