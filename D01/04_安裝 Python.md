# 安裝 Python


## Windows

1. 官方  [下載](https://www.python.org/downloads/windows/)

    ![](images/img_01.png)

2. 以系統管理員安裝

    ![](images/img_02.png)

3. 務必勾選添加路徑，然後選擇客製化安裝

    ![](images/img_03.png)

4. 點擊 `NEXT`

    ![](images/img_04.png)

5. 勾選 `Install for all users`，就是因為要在這裡確認 `路徑中沒有中文`，所以選擇客製化安裝。

    ![](images/img_05.png)

6. 安裝完成，關閉即可。

    ![](images/img_06.png)

## 回到終端機

1. 查詢版本
```bash
python --version
```

2. 就會是剛剛安裝完成的版本。

    ![](images/img_07.png)
