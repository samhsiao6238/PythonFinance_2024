#

## 使用 Homebrew 安裝 neo4j

使用 brew 安裝 neo4j
```bash
brew install neo4j
```

啟動
```bash
neo4j start
```

## 建立本地專案

新增 `Local DBMS`，點擊 `Next`。
![](images/img_10.png)

使用預設的命名 `Graph DBMS`，自訂密碼為 `sam112233`，選擇最新版本 `5.19.0`，預設也會是最新版 `latest`，然後點擊建立 `Create`。
![](images/img_11.png)

需要等待一下，進度條會顯示進度。
![](images/img_12.png)

完成時點擊 `Start`。
![](images/img_13.png)

會自動處理衝突，點擊修理 `Fix configuration`。
![](images/img_14.png)
```json
bolt: 7687 → 7689
http: 7474 → 7475
discovery: 5000 → 5001
cluster.raft: 7000 → 7001
routing: 7688 → 7689
```

還是衝突，就再點擊一次。
![](images/img_15.png)
