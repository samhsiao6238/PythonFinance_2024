## 連續期貨合約

_Continuous Futures，當期貨合約過期後，原合約無法再查詢數據，但可以使用 `TXFR1` 和 `TXFR2` 來取得近月和次月的連續合約數據。_

<br>

1. 逐筆資料 (Ticks)。

    ```python
    # 逐筆資料
    ticks = api.ticks(
        contract=api.Contracts.Futures.TXF.TXFR1, 
        date="2020-03-22"
    )
    ticks
    ```

<br>

2. K 線數據 (Kbars)。

    ```python
    # K 線數據
    kbars = api.kbars(
        contract=api.Contracts.Futures.TXF.TXFR1,
        start="2023-01-15", 
        end="2023-01-16", 
    )
    kbars
    ```
