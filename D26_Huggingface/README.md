## 介紹

1. `維基百科`：`Hugging Face` 是一家美國公司，專門開發用於建立機器學習應用的工具。該公司的代表產品是其為自然語言處理應用建立的 `transformers` 庫，以及允許使用者共享機器學習模型和資料集的平台。

2. 2023 年開始與 AWS 進行合作，可結合 AWS 服務來使用 Huggingface。
