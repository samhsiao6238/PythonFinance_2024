# 關於指令 tree

_在 Linux 是預設的_

## 安裝 Homebrew

1. 查詢版本
```bash
brew -v
```
2. 假如要安裝 Homebrew
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

3. 安裝。
```bash
brew install tree
```

## brew 報錯

1. 修理。
```bash
brew tap --repair
```

2. 更新。
```bash
brew update
```

3. 檢查。
```bash
brew doctor
```
