# 安裝 Python

_前往 [官網](https://www.python.org/) 進行安裝_

<br>

## Windows

1. 在下載的頁籤中選擇 Windows。

    ![](images/img_32.png)

<br>

2. 官網 [下載](https://www.python.org/downloads/windows/) 。

    ![](images/img_21.png)

<br>

3. 以系統管理員身份執行安裝。

    ![](images/img_22.png)

<br>

4. 務必勾選添加路徑，然後選擇客製化安裝。

    ![](images/img_23.png)

<br>

5. 點擊 `NEXT` 。

    ![](images/img_24.png)

<br>

6. 勾選 `Install for all users`，就是因為要在這裡確認 `路徑中沒有中文`，所以選擇客製化安裝。

    ![](images/img_25.png)

<br>

7. 安裝完成，關閉即可。

    ![](images/img_26.png)

<br>

_回到終端機_

8. 查詢版本。

    ```bash
    python --version
    ```

<br>

9. 就會是剛剛安裝完成的版本。

    ![](images/img_27.png)


<br>
